# NexusAI 更新日志

---

## V1 — DocuMind

提供 RAG、MCP、Memory、Agent 四大核心组件：

| 组件 | 目录 | 说明 |
|------|------|------|
| **RAG** | `core/rag/` | 向量检索 + LLM 生成，封装完整 RAG 流程 |
| **MCP** | `core/mcp/` | Model Context Protocol 集成，连接外部工具服务 |
| **Memory** | `core/memory/` | 会话管理，对话上下文存储与检索 |
| **Agent** | `core/agent/` | 意图路由 + 智能体编排，LLM 驱动的意图分发 |

---

## V2 — aiops_agent

在 V1 基础上新增与重构：

### 重构

- **RagComponentImpl**: 提示词配置化（新增 `rag.system-prompt` / `rag.empty-response` 配置项，优先级：方法参数 > 配置文件 > 默认值）
- **DocumentComponentImpl**: 重构为使用 StorageAdapter，存储层可插拔（不再绑死 Redis）

### 新增

- **DocumentStorageAdapter**: 存储适配接口，抽象文档的增删查操作
- **QdrantStorageAdapter**: 基于 Qdrant 向量库的实现（`@ConditionalOnBean` 自动装配）
- **RedisStorageAdapter**: 基于 Redis 的实现（`@ConditionalOnBean` 自动装配）
- **prometheus 组件** (`core/tools/prometheus/`):
  - `PrometheusConfig`: 配置类（url / queryDuration / queryStep）
  - `PrometheusService`: 封装 Prometheus HTTP API，含 WebClient UriBuilder 踩坑记录
  - `PrometheusTools`: `@Tool` 注解工具，PromQL 模板可通过构造函数自定义
